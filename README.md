### Mac
<img width="260" height="235" alt="image" src="https://github.com/user-attachments/assets/f83e3958-f168-4a23-b451-a9784119e142" />

Mac版本打开软件遇到上图警告，需要执行`xattr -r -d com.apple.quarantine`

```sh
xattr -r -d com.apple.quarantine "/Applications/Anylink Client.app"
```
